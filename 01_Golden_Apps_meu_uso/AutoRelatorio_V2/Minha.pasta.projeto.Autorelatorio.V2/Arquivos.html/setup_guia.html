Guia de Instalação
AutoRelatorio V2 · PC novo ou formatado
0 / 12 concluídos
01
📦
Requisitos do sistema
Python e Node.js instalados globalmente
▼
✓
Instalar Python 3.10 ou superior

Baixe em python.org e execute o instalador.

Obrigatório: na primeira tela do instalador, marque a caixinha "Add Python to PATH" antes de clicar em Install. Sem isso o terminal não encontra o Python.

Confirme a instalação:

python --version
✓
Instalar Node.js 20 LTS ou superior

Baixe o LTS em nodejs.org. O npm já vem junto.

Confirme a instalação:

node --version
npm --version
02
📁
Obter o código
Clonar ou copiar os arquivos do projeto
▼
✓
Copiar ou clonar a pasta do projeto

O projeto deve estar em algum local do seu computador. O caminho pode ser qualquer um — anote onde ficou, você vai precisar.

Exemplo: C:\AutoRelatorio_V2

Se veio via Git:

git clone <repositório> C:\AutoRelatorio_V2
03
⚙️
Preparar o Backend (Python)
Ambiente virtual e dependências da API
▼
✓
Navegar até a pasta do backend

Abra o PowerShell ou Prompt de Comando e execute:

cd C:\AutoRelatorio_V2\APP\backend
✓
Criar o ambiente virtual Python

O ambiente virtual isola as bibliotecas do projeto para não conflitar com outros programas.

python -m venv .venv
✓
Ativar o ambiente virtual
.\.venv\Scripts\activate
Como saber se funcionou: o terminal vai mostrar (.venv) no início da linha, antes do caminho.
✓
Instalar as dependências Python
pip install -r requirements.txt
O que instala: FastAPI, uvicorn, python-docx, Pillow, e outras bibliotecas listadas no requirements.txt.
04
🖥️
Preparar o Frontend (Node.js)
Dependências da interface React/Next.js
▼
✓
Navegar até a pasta do frontend
cd C:\AutoRelatorio_V2\APP\frontend
✓
Instalar os pacotes Node
npm install
Aguarde: esse processo baixa milhares de sub-arquivos (node_modules). Pode demorar 2–5 minutos dependendo da internet. Normal.
05
🚀
Rodar o aplicativo
Uso diário — um único comando
▼
✓
Opção A — Duplo clique no run.bat

Basta dar duplo clique no arquivo run.bat na raiz do projeto. O terminal abre automaticamente com tudo configurado.

✓
Opção B — Pelo terminal
cd C:\AutoRelatorio_V2
python run.py
O que acontece: o orquestrador limpa as portas 3000 e 5000 (se presas), sobe o backend FastAPI, aguarda ele responder, então sobe o frontend Next.js. Quando aparecer SISTEMA PRONTO, acesse http://localhost:3000.
✓
Encerrar o sistema

Pressione Ctrl+C no terminal. O orquestrador encerra backend e frontend juntos, sem deixar processos presos.

✦
Instalação completa
Para o uso diário: run.bat ou python run.py
Interface em http://localhost:3000
