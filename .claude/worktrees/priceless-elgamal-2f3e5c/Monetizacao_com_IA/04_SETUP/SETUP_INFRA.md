# 🛠️ Guia de Setup e Infraestrutura Técnica

Este é o guia de instalação das ferramentas necessárias para rodar o motor de monetização com a velocidade do Guilherme Franklim.

## 1. Ambiente de Desenvolvimento (O Motor)
Para rodar as automações de criação de página:
*   **VS Code:** Editor principal.
*   **Claude Code (Extensão):** Essencial para o Agente @Franklim-MVP construir as páginas.
*   **Node.js & NPM:** Necessário para rodar o Vercel CLI.

## 2. Hospedagem e Deploy (Custo Zero)
*   **GitHub:** Crie uma conta para versionar seus arquivos.
*   **Vercel:** Conecte ao GitHub. O deploy é automático: cada `git push` atualiza seu site no ar em segundos.
*   **Comando Útil:** `npx vercel` no terminal para subir o site sem interface.

## 3. Ativos Visuais (I.A Designer)
*   **Grok (X Premium) ou Leonardo.ai:** Para gerar imagens profissionais.
*   **Pollinations.ai:** Gerador de imagens via URL (grátis) para testes rápidos na Landing Page.

## 4. Meta Ads (A Fonte de Tráfego)
*   **Configuração do Pixel:** Deve estar instalado na Landing Page (Head) e na Kiwify.
*   **Estratégia de Lance:** Verifique se sua conta tem o "Limite de Lance" liberado. Se for conta nova, rode R$ 100 em "Maior Volume" primeiro para aquecer.

## 5. Checkout (O Caixa)
*   **Kiwify:** Configure o "Checkout Personalizado". Use o modo "Global" para aceitar moedas LATAM e conversão automática.
