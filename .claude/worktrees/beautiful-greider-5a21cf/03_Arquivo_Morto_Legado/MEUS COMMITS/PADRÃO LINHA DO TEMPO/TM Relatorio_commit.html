🌐 Migração para Arquitetura Web (Vite/React + Flask)

Concluído

Transformação completa do desktop (CustomTkinter) para uma aplicação web nativa:

  • ✓Frontend Vite + React: UI reconstruída com Componentes, mantendo a identidade visual Premium Dark.
  • ✓Backend Flask: Servidor local servindo a API responsável pelo escaneamento de pasta e geração DOCX.
  • ✓Preview Otimizado: O módulo de pré-visualização agora roda no navegador, com paginação fluída e thumbnails reais (base64) criados pelo backend.
  • ✓Orquestração: run_app.bat atualizado para instalar dependências e iniciar ambos os servidores automaticamente de forma invisível/minimizada.

🚀 Evolução para Next.js 15 + FastAPI (Arquitetura Premium)

Concluído

Salto tecnológico para uma infraestrutura de ponta, unindo a robustez do Python com a modernidade do Next.js:

  • ✓Frontend Next.js 15: Interface reconstruída do zero com o App Router, oferecendo transições instantâneas e melhor SEO interno.
  • ✓Design Glassmorphism: Estética ultra-moderna com efeitos de desfoque, transparências e animações via Framer Motion.
  • ✓Backend FastAPI: Migração total de Flask para FastAPI, garantindo respostas sub-milissegundos e documentação automática Swagger.
  • ✓Terminal Integrado: Novo painel de logs com estilo de console hacker para acompanhamento detalhado da geração do Word.
  • ✓Workflow Unificado: Script run.bat otimizado para orquestrar o motor Python e o servidor Next.js simultaneamente.

✨ Polimento de UX Premium & Download Nativo

Concluído

Elevando a interface do Next.js para um nível superior e simplificando a jornada do usuário no fim da geração:

  • ✓Download Inteligente: Criação de endpoint /api/download no FastAPI para servir o .docx direto do navegador.
  • ✓Integração UI/UX: Um novo botão de Download Flutuante que aparece automaticamente ao final do processo, evitando que o usuário precise abrir a pasta de saída manualmente.
  • ✓Header Flutuante: Cabeçalho fixo (sticky) com backdrop-blur-xl e glassmorphism suave, mantendo todos os botões acessíveis sempre.
  • ✓Cards e Sombras: Profundidade redesenhada usando multicamadas (20px, 40px) de sombra e micro-interações (borda suave e Glow no hover).
  • ✓Botões Vibrantes: Tailwind gradient e pulse nas ações principais, direcionando o foco do usuário visualmente.
  • ✓Footer Reinventado: Tipografia tracking-wide elegante e status de versão destacado.
