# 📱 Estratégia: TikTok Shop Automation (Faceless & Global)

Transforme o Claude Code em um robô que trabalha 24h minerando produtos e gerando vídeos para o TikTok Shop.

## 1. Ferramentas de Mineração
*   **A Viral:** Foco em roupas femininas e tendências rápidas (Brasil).
*   **Calodata:** Inteligência de dados para TikTok Shop (Gringa/Global).

## 2. O Fluxo "Um Clique"
1.  **Instrução de Projeto (Project Instructions):** Configure o Claude Code com a memória da "A Viral".
2.  **Navegação Automática:** O Claude abre a Viral, seleciona o "Produto 1" com boa visibilidade e extrai os detalhes.
3.  **Geração Visual (Gemini/V03):** 
    *   O Claude gera o prompt: *"Mulher jovem fazendo mirror selfie segurando o celular, mostrando [PRODUTO] em um cenário de quarto iluminado."*
    *   O Gemini gera o vídeo (UGC).

## 3. Postagem e Escala
*   **Edição:** Recorte para 9:16 no Capcut.
*   **Escala:** Repita o processo para 5-10 produtos por dia.

## 4. Diferencial Global (DTS Global)
*   Para rodar na gringa, o Agente @TikTok-Automator ajusta o idioma dos prompts e pesquisa produtos no Calodata USA.
