SEUS 3 MODOS (ESCOLHA 1)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[1] APENAS NOTIFICAR (Máximo Controle)
    📁 Mudança detectada
    → Você decide o que fazer

[2] PERGUNTAR SEMPRE (Recomendado)
    📁 Mudança detectada
    Deseja fazer staging? [Sim/Não]
    → Balanço entre controle e automação

[3] AUTO-COMMIT (Máxima Automação)
    📁 Mudança detectada
    🤖 Auto-commit + push (sem perguntar)
    → Máxima automação

Configure com: /tm-commit settings

COMANDOS DISPONÍVEIS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

/tm-commit status     Ver mudanças
/tm-commit add        Fazer staging
/tm-commit commit     Criar commit
/tm-commit push       Fazer push
/tm-commit sync       Pull + push completo
/tm-commit log [n]    Ver últimos n commits
/tm-commit clean      Limpar branches antigas
/tm-commit help       Ver ajuda

COMPORTAMENTO AUTOMÁTICO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔄 INÍCIO DA SESSÃO (Session-Start)
   → Sincroniza repositório
   → Mostra status

📁 AO CRIAR ARQUIVO (File-Created)
   → Detecta mudança
   → Executa sua opção escolhida (1, 2 ou 3)

📋 FIM DA SESSÃO (Session-End)
   → Mostra resumo de mudanças
   → Oferece 3 opções: [Sim] [Não] [Revisar]
