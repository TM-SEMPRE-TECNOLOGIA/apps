⚙️ Como Instalar

1

Opção 1: Como Skill do Cowork (Recomendado)

Copie a pasta tm-commit-skill para:

C:\Users\[seu-user]\AppData\Roaming\Claude\skills\

Ou procure por "tm-commit" na aba Skills do Cowork Dashboard

2

Opção 2: Direto no Chat

Basta invocar em qualquer sessão:

/tm-commit status

O skill é auto-carregado no início de cada sessão

3

Opção 3: Via NPM (Se Configurado)

npm install tm-commit-skill npx tm-commit status

🎯 Funcionalidades

🔄 Automação Git

Detecta mudanças, cria commits e faz push automático para GitHub

✓ Validação

Valida padrão de commits e regras de segurança antes de fazer push

📊 Monitoramento

Monitora 8 pastas do repositório e detecta mudanças em tempo real

🔒 Segurança

Bloqueia credenciais, secrets e arquivos suspeitosos automaticamente

📝 Conventional Commits

Força uso de Conventional Commits para histórico limpo e rastreável

📈 Relatórios

Gera relatórios de commits, arquivos alterados e sincronização

⚡ Comandos Rápidos

/tm-commit status # Ver mudanças /tm-commit push # Fazer push /tm-commit sync # Pull + Push /tm-commit log 5 # Ver 5 últimos commits /tm-commit help # Mostra ajuda
