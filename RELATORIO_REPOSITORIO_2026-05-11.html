Context7 — Integração com Repositório

Context7 é um serviço de documentação contextual que pode ser integrado ao seu repositório de 3 maneiras:

1. Via package.json
Declare dependências no package.json com versões exatas. Context7 indexa automaticamente.
2. Via .context7rc ou CLAUDE.md
Arquivo de configuração que aponta para repositórios de código, documentação e referencias. Context7 aprende a estrutura do seu projeto.
3. Via Context7 Query Tool
Use mcp__query-docs com seu libraryId para consultar docs de forma contextual durante sessões.

Próximo passo: Criar arquivo .context7rc.json na raiz do repositório para mapear seus agentes, skills, design system e documentação técnica.
