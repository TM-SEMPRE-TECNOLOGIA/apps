# Referências Adicionais de Inspiração para Web Design (via Reddit r/webdev)

Esta seção contém referências valiosas extraídas de uma discussão na comunidade r/webdev sobre onde encontrar inspiração de alta qualidade para design de sites.

## Sites de Inspiração e Referência

| Recurso | Link | Descrição |
| :--- | :--- | :--- |
| **Landing Love** | [landing.love](https://www.landing.love/) | Exibe sites com animações criativas e interações únicas. |
| **Land-Book** | [land-book.com](https://land-book.com/) | Uma das melhores galerias de landing pages, com filtros por diversas categorias. |
| **Built For Mars** | [builtformars.com](https://builtformars.com/ux-glossary) | Focado em UX de alta qualidade, inclui um glossário de UX excepcional. |
| **Webflow Portfolio Showcase** | [webflow.com/made-in-webflow](https://webflow.com/made-in-webflow/portfolio/popular) | Biblioteca de projetos criados no Webflow, incluindo templates clonáveis. |
| **UI Patterns** | [ui-patterns.com](https://ui-patterns.com/) | Excelente para entender princípios de UI/UX através de padrões recorrentes. |
| **Call to Idea** | [calltoidea.com](http://www.calltoidea.com/) | Único por oferecer inspiração para componentes específicos (rodapés, formulários, etc.). |
| **Nicely Done Club** | [nicelydone.club](https://nicelydone.club/) | Focado em UX e princípios de conversão, ideal para entender o design funcional. |
| **Landing Folio** | [landingfolio.com](https://www.landingfolio.com/) | Galeria específica para landing pages com categorização extensa. |
| **Saas Landing Page** | [saaslandingpage.com](https://saaslandingpage.com/) | Inspiração focada especificamente em produtos SaaS. |
| **Lapa Ninja** | [lapa.ninja](https://www.lapa.ninja/) | Curadoria de landing pages de alta performance. |
| **Page Collective** | [pagecollective.com](https://pagecollective.com/) | Outra excelente galeria focada em design de landing pages. |
| **One Page Love** | [onepagelove.com](https://onepagelove.com) | Especializado em sites de página única, com muitas categorias. |
| **Psychology of Conversions** | [psychologyofconversions.com](https://www.psychologyofconversions.com/) | Um diretório compilado por um designer com mais de 450 referências categorizadas. |
