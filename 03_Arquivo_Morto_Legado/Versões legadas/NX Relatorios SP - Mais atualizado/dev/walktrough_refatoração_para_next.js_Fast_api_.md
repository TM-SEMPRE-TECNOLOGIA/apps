# Walkthrough: Nova Era do Gerador de Relatórios

Transformamos o antigo sistema React/Vite/Flask em uma aplicação moderna com **Next.js 15** e  **FastAPI** , focada em performance e estética premium.

## O Que Mudou?

* **Frontend** : Migrado para Next.js (App Router). Agora com design  **Glassmorphism** , animações suaves (Framer Motion) e ícones elegantes (Lucide React).
* **Backend** : Convertido de Flask para  **FastAPI** . Mais rápido, tipado com Pydantic e pronto para escalabilidade.
* **Experiência do Usuário** : Logs em tempo real com estilo de terminal, pré-visualização da estrutura de pastas melhorada e seletor de pastas nativo do Windows integrado.

## Como Executar

Para facilitar sua vida, criei um script que inicia tudo de uma vez:

1. Vá até a pasta raiz: `C:\Users\thiag\TM-MEUS-APPS\Organizar\TM Relatorio - REACT APP`
2. Dê dois cliques no arquivo  **`run.bat`** .
3. Aguarde o backend e o frontend iniciarem.
4. O navegador abrirá automaticamente ou você pode acessar: **[http://localhost:3000](http://localhost:3000/)**

## Tecnologias Utilizadas

* **Next.js 15** : Framework React para produção.
* **Tailwind CSS** : Estilização moderna e rápida.
* **FastAPI** : Backend Python de alta performance.
* **Uvicorn** : Servidor ASGI para Python.
* **Framer Motion** : Animações de interface.
