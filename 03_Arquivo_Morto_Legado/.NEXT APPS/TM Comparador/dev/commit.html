🚀 Início do Projeto

Setup

Criação do TM Comparador — ferramenta para comparar itens de relatórios fotográficos (DOCX) com orçamentos (XLSX).

  • ✓Definição do escopo: comparar quantitativos entre documentos
  • ✓Setup do ambiente Python com venv
  • ✓Dependências: pandas, openpyxl, python-docx
  • ✓Estrutura de pastas e requirements.txt
Python 3.12 venv pandas

🧠 Motor de Comparação

Core

Implementação completa da lógica de análise e comparação em comparer.py.

  • ✓load_report_items() — Extrai itens de tabelas "Itens" em DOCX
  • ✓load_budget_items() — Lê aba "RAT" do XLSX com detecção automática de colunas
  • ✓compare_items() — Cruza dados e classifica: Compatível, Divergente, Só Orçamento, Só Relatório
  • ✓write_outputs() — Gera log TXT e CSV timestampados
  • ✓Detecção inteligente de header row no Excel (scan de 80 linhas)
  • ✓Normalização de códigos e quantitativos com regex robusto
python-docx openpyxl pandas regex dataclasses

🖥️ Interface Desktop (Tkinter)

v1.0

Primeira versão estável com interface gráfica nativa em Tkinter.

  • ✓Seletores de arquivo para Relatório (.docx) e Orçamento (.xlsx)
  • ✓Tabela de resultados com Treeview (Item, Qtd. Relatório, Qtd. Orçamento, Status)
  • ✓Barra de resumo com totais por status
  • ✓Botão "Abrir pasta de saída" para acessar logs e CSV
  • ✓run_app.bat para execução com clique duplo
tkinter ttk.Treeview filedialog

⚡ Migração para FastAPI

Backend

Backend REST criado para servir a nova interface web, reutilizando 100% da lógica de comparer.py.

  • ✓/api/dialog/file — Dialogo nativo para seleção de arquivos
  • ✓/api/compare — Executa comparação e retorna JSON com resultados + resumo
  • ✓/api/download — Serve logs .txt e .csv para download
  • ✓/api/outputs — Lista histórico de comparações anteriores
  • ✓/api/upload — Aceita upload de arquivos via drag & drop
  • ✓CORS configurado para desenvolvimento local
FastAPI uvicorn pydantic python-multipart
