# TM Extrator 🚀

Repositório organizado e padronizado para extração técnica de dados. Este projeto utiliza o ecossistema Premium TM para garantir eficiência e organização no desenvolvimento.

## 📂 Estrutura de Pastas

- **`/app`**: Contém a aplicação principal desenvolvida em **Vite/React**. É aqui que reside a interface do usuário e a lógica de processamento frontal.
- **`/dev`**: Ferramentas e logs de desenvolvimento. Contém o `commit.html` para rastreamento de progresso.
- **`/documentos`**: Arquivos de referência, documentos `.docx` para testes e registros de progresso.
- **`/legado`**: Versões anteriores, rascunhos de design e iterações descartadas.
- **`/scripts`**: Scripts de suporte, utilitários Python e ferramentas de linha de comando.

---

## 🚀 Como Executar

### Forma Automática
Para iniciar a aplicação rapidamente, utilize o arquivo na raiz:
1.  Execute o arquivo `rodar.bat`.
2.  O script entrará automaticamente na pasta `/app` e iniciará o servidor de desenvolvimento.

### Execução Manual
Caso prefira rodar via terminal:
1.  Abra o terminal na raiz do projeto.
2.  Navegue até a pasta do aplicativo:
    ```bash
    cd app
    ```
3.  Instale as dependências (necessário apenas na primeira vez):
    ```bash
    npm install
    ```
4.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
