Onde Configurar a IA 🧠

Guia rápido do desenvolvedor para o APP do TM Chat

🚀 Como Rodar a Aplicação

Para iniciar o servidor de desenvolvimento do TM Chat e testar no seu navegador, siga os passos abaixo:

Passos para iniciar:

  1. Abra um terminal na pasta do projeto (nextjs-chat).
  2. Caso seja a primeira vez rodando o projeto, instale as dependências com o comando:
    npm install
  3. Inicie o servidor de desenvolvimento rodando o seguinte comando:
    npm run dev
  4. Acesse no seu navegador o endereço:
    http://localhost:3000

Dica: Para parar o servidor a qualquer momento, volte no terminal onde ele está rodando e pressione CTRL + C.

🤖 Como trocar o Modelo de IA

A configuração do modelo que responde às mensagens é dividida em dois lugares, dependendo de como você quer controlar:

Configuração Base (Onde deve mudar primeiro):

O arquivo .env.local na raiz do projeto nextjs-chat.

Nele você encontra a linha: DEFAULT_MODEL="openrouter/google/gemini-2.0-flash-exp:free"

Altere essa linha para o nome de qualquer modelo suportado pelo OpenRouter (ex: "google/gemini-pro", "anthropic/claude-3-haiku" ou "meta-llama/llama-3-8b-instruct").

Onde o Código consome isso (Backend API):

No arquivo src/app/api/chat/route.ts (no servidor).
Lá dentro existe uma linha que diz:
const model = process.env.DEFAULT_MODEL || "google/gemini-2.0-flash-exp:free";

Se a variável no .env sumir, ele vai usar o nome do modelo que está escrito aqui como backup.

🎓 Como Treinar o seu Bot (Comportamento)

Se você quiser que o "TM - Tecnologia Sempre" aja de uma maneira específica (ex: ser sarcástico, focar apenas em programação, ou falar como um pirata), você faz isso no mesmo lugar onde a API é chamada!

O Arquivo Chave (System Prompt):

Abra o arquivo src/app/api/chat/route.ts. Encontre o trecho de código parecido com este: 

messages: [
  {
    role: "system",
    content: "Você se chama TM..." // ← É AQUI QUE A MÁGICA ACONTECE!
  },
  ...messages,
],

A mensagem com role: "system" é a "alma" do bot. É o primeiro comando oculto enviado pra ele. Mude o content para descrever exatamente como você quer que ele seja!

Ajustando a "Criatividade" (Temperature):

Você pode controlar se a IA é mais precisa/robótica ou mais aleatória/criativa no mesmo bloco onde escolheu o model.

const completion = await openai.chat.completions.create({
  model: model,
  temperature: 0.7, // 0 = Frio/Exato | 1 = Muito Criativo | Até 2 (Loucura total)
  messages: [...]
});

Muitos modelos de programação funcionam bem em 0.2, e para escrita/conversa criativa as pessoas usam 0.7 a 0.9.

✨ Resumo: Quero mudar de IA agora!

Siga estes passos exatos:

  1. Abra o arquivo TESTE/nextjs-chat/.env.local no seu editor.
  2. Mude a linha do `DEFAULT_MODEL` para o novo modelo que você achou no site do OpenRouter (O nome dele costuma ficar no topo da página de modelos deles, exemplo: meta-llama/llama-3-8b-instruct).
  3. Salve o arquivo.
  4. Pare o servidor no terminal clicando nele e apertando CTRL + C.
  5. Rode o servidor de novo: npm run dev (Variáveis .env só são detectadas quando o servidor reinicia!).
