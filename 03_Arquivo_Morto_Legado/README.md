# 📦 03_Arquivo_Morto_Legado

> **Museu Histórico & Portfólio de Testes**

Este é o Cérebro Histórico. Todo o código escrito, refatorado, congelado ou quebrado durante as madrugadas de estudos está depositado aqui. **Este repositório é apenas para leitura e preservação**.

## 🛡️ O Valor do Caos
- Cada pasta dentro deste repositório representa uma linha do tempo anterior que ajudou a pavimentar as versões Golden (01 e 02). 
- Erros de TypeScript, deploys com falha (`NX Relatórios SP Up`), experiências abandonadas de UIs antigas habitam esses diretórios.
- É a prova digital do amadurecimento como Engenheiro de Software Full-Stack (Portfólio).

## ⚠️ Manutenção
- Este repositório é livre de pacotes massivos. Todas as instâncias de `node_modules`, `.venv` e `.next` foram obliteradas via scripts em terminal PowerShell para garantir que a nuvem fique leve, mantendo apenas a riqueza dos códigos-fontes (`.tsx`, `.py`).

O Cérebro do Assistente Virtual (.jarvis) continua inserido como backup neste repositório.
