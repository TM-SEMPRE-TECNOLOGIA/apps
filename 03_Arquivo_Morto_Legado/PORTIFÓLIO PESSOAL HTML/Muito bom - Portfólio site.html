Análise de Módulos Críticos

Um mergulho profundo na lógica de negócios e estruturação de dados de sistemas desenvolvidos recentemente.

O Hub Unificado (TM Ecossistema)

Uma plataforma centralizadora que consolida 9 ferramentas críticas corporativas em uma única interface. A arquitetura exige um roteamento dinâmico sem fricção e isolamento de estado por sub-aplicação.

  • Controle de acesso unificado e sistema global de favoritos mantido via Zustand com persistência local.
  • Containerização frontend das ferramentas: Ordens de Serviço, Comparador de Dados Financeiros, e Gerador Documental SP.
  • Comunicação assíncrona multiplexada com múltiplos endpoints FastAPI rodando processos intensivos de I/O em background.
store/useHubStore.ts TypeScript 
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

interface HubState {
  activeModule: string | null;
  favorites: string[];
  setActive: (moduleId: string) => void;
  toggleFavorite: (appId: string) => void;
}

export const useHubStore = create<HubState>()(
  persist(
    (set) => ({
      activeModule: null,
      favorites: [],
      setActive: (id) => set({ activeModule: id }),
      toggleFavorite: (id) => 
        set((state) => ({
          favorites: state.favorites.includes(id) 
            ? state.favorites.filter(x => x !== id) 
            : [...state.favorites, id]
        }))
    }),
    { name: 'tm-hub-storage' }
  )
);

Engenharia Documental (TM Extrator & DOCX)

O coração da automação inicial que evoluiu para o 'TM Relatório SP'. Sistemas Python capazes de ler metadados, manipular tabelas word, inserir imagens em templates complexos e recalcular somatórios matemáticos em tempo real direto nos nós XML do documento Word.

  • Manipulação via `python-docx` para ancoragem precisa de imagens mantendo "Vistas Amplas" em alta resolução sem quebrar a paginação.
  • Motor de cálculo acoplado (Subtotal, Descontos) injetando strings formatadas e destacando dados críticos em vermelho.
  • Regex avançado para parsing de dezenas de colunas do Excel transformando-as instantaneamente em Parágrafos DOCX consolidados.

def insert_financial_table(doc, data_frame):
    # Parse DF and inject into template
    table = doc.add_table(rows=len(data_frame)+1, cols=4)
    # ... header definitions ...
    
    total = 0.0
    for i, row in data_frame.iterrows():
        # Injecting item logic
        cell = table.cell(i+1, 3)
        val = float(row['Value'])
        total += val
        
        # Apply conditional formatting
        p = cell.paragraphs[0]
        run = p.add_run(f"R$ {val:,.2f}")
        if val < 0:
            run.font.color.rgb = RGBColor(255, 0, 0)
            run.bold = True
            
    append_summary_row(table, total)
    return doc
